# 🚀 Lens Protocol Project

A modern web-based project built to explore and implement core functionalities of Lens Protocol.

---

## 📌 About The Project

This project is created to understand and implement the concepts of decentralized social graph using Lens Protocol.

It demonstrates:
- Smart contract interaction
- Web3 integration
- API usage
- Frontend integration
